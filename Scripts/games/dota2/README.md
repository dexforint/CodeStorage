# Запуск

1. Открой Steam, перейди в библиотеку, нажми правой кнопкой на Dota 2 → Свойства → вкладка "Общее". В поле "Параметры запуска" добавь:

```
-gamestateintegration
```

2. Нужно добавить конфиг-файл в директорию `steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\`. Файл должен использовать шаблон имени `gamestate_integration_*.cfg`.

3. Запустите сервер dota2_server.py
4. Запустите игру
