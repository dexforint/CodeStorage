# Описание проекта

Давай создадим веб-приложение на FastAPI. Данное приложение представляет собой небольшой SaaS проект по генерации изображений с прозрачны фоном.

Вот примерно как я его вижу:

- Пользователь авторизуется на сайте. Авторизация может происходить как через Email, так и через Google и Github.
- Каждый день пользователю даётся квота в 100 кредитов.
- Пользователь может потратить эти кредиты на генерацию изображений с прозрачным фоном.
- Стоимость генерации зависит от модели генерации.
- Пользователь вводит свой промпт для генерации. А далее:
  - Задача по генерации ставится в очередь. Чем больше кредитов у пользователя, тем приоритнее его задача (быстрее выполнится).
  - Выполняется задача примерно так: сначала генерируется изображение с помощью диффузионной модели, а затем с помощью дополнительной нейросетевой модели удаляется фон у сгенерированного изображения.
  - Сам процесс обработки должен быть виден пользователю.
- Пользователь должен иметь возможность посмотреть все его задачи с их статусами ("В очереди", "Генерация" (как вариант "Генерация: X%"), "Обрезка фона", "Готово").
- Проект должен быть хорошо оптимизирован, чтобы обрабатывать много запросов на GPU и CPU.
