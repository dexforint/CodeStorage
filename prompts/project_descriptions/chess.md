# Описание проекта

Давай создадим веб-приложение на FastAPI. Данное приложение представляет собой сайт для игры в шахматы (наподобие lichess.org).

Вот примерно как я его вижу:

- Пользователь авторизуется на сайте. Авторизация может происходить как через Email, так и через Google.
- Пользователь выбирает режим игры: 1min, 3min, 30min.
- Пользователь нажимает кнопку "Найти игру". Для пользователя подбирается другой игрок, который сейчас тоже ищет игру в данном режиме и который примерно соответствует рейтингу первого игрока.
- Когда два игрока найденты - они начинают играть между собой.
- У игрока есть возможность:
  - Сдаться.
  - Написать другому игроку.
- Если игрок вышел или у него отлетел интернет, то он имеет 10 секунд, чтобы зайти, иначе ему будет засчитано поражение.
- Если игрок обновляет страницу, то у него должно сохраниться состояние игры, время и все комментарии.
- После окончания игры рейтинги игроков обновляются.
- Игроки могут играть несколько партий одновременно.
- На сайте должен быть лидерборд.
