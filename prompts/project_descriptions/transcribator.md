# Описание проекта

Давай создадим веб-приложение на FastAPI. Данное приложение представляет собой небольшой голосовой чат с транскрибацией речи.

Вот примерно как я его вижу:

- Пользователь заходит на сайт. При входе указывает имя.
- Затем пользователя кидает в общим голосовой чат.
- В этом голосовом чате пользователь разговаривает с другими пользователями.
- В процессе их разговора на странице чата отображается транскрибация их речи, при чём с разделением на говорящих.
- В процессе разговора должна отображаться аудио-визуализация (audio waveform / waveform visualization).
- Проект должен быть хорошо оптимизирован, чтобы обрабатывать много запросов на GPU и CPU.
